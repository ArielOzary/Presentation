# Web.AutoLog

Web application, stack:

- React
- Vite
- ReactQuery
- Zustand
- React Hook Form + Zod
- SCSS

### Startup

1. Create `.env` file

```bash
# see full example in template.env file
VITE_API_URL=http://localhost:3000/
```

2. Run development

```bash
$ yarn dev
```
